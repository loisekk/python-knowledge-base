## 📖 About

This repository is my personal Python learning journal — built from the ground up, covering every core concept of the language. Each file and folder represents a chapter in my journey from writing the first `print()` to understanding object-oriented design patterns.

> Every line here was written by hand. No copy-paste, no shortcuts — pure practice.

## 🔥 Highlight Programs

### 🎮 Number Guessing Game — `seventh.py` / `game.py`
A fully functional CLI game with input validation, range selection, and attempt counter.

### 📁 Hi-Score Tracker — `sixth.py`
Persistent score storage using File I/O — reads, compares, and writes high scores across sessions.

### ✨ Star Pattern Printer — `fourth.py`
Generates pyramid, hollow square, and inverted triangle patterns using nested loops.

### ✊ Rock Paper Scissors — `fifth.py`
Dictionary-based game logic with randomized computer choice and result detection.

### 🔢 Bubble & Selection Sort — `seventh.py`
Manual implementations of classic sorting algorithms — no built-in sort used.

## 🚀 Getting Started

**Requirements:** Python 3.x (no external libraries required for most files)

```bash
# Clone the repo
git clone https://github.com/loisekk/python-knowledge-base.git
cd python-knowledge-base

# Run any file
python source_code/seventh.py
```

> Some files use `numpy` and `random` (standard library). Install numpy if needed:
> ```bash
> pip install numpy
> ```

## 🧠 Learning Philosophy

This repo follows a **build-first, understand-later** approach:
- Concepts are explored through real programs, not just syntax demos
- Each file builds on the previous one
- Problems are solved manually before using built-ins
- Comments serve as learning notes, not just documentation
